15  Markdown Basics

Markdown files contain basic formatting capabilities. The use of the # followed by text creates a heading. Using two or more # symbols will create subheadings based on the number of #. A text is italicized by surrounding the text with one asterisk (*italicized*). A text is boldfaced by surrounding it with 2 asterisk (**boldfaced**). A text is strikethrough by surrounding the text with 2 tildes (~~strikethrough~~). A text withsuperscript is created by surrounding a word with caret (^superscript^). A text withsubscript is created by surrounding a word with 1 tilde (~subscript~).

To create an unordered list, use the - symbol at the beginning of each line. To create a sub-item, press the tab button, then the - symbol. Repeat this method for further sub-items.

Syntax Output 
- First Item
- Second Item
   - First Sub-Item
       - First Sub-Sub Item
           - First Sub-Sub-Sub Item
  • First Item
  • Second Item
    • First Sub-Item
      • First Sub-Sub Item
        • First Sub-Sub-Sub Item

To created an ordered list, type the number followed by a period for each line. To create sub-lists, press the tab button twice and order them appropriately.

Syntax Output 
1. First Item
2. Second Item
   a. First Sub-Item
       i. First Sub-Sub-Item
       ii. Second Sub-Sub-Item
  1. First Item
  2. Second Item
    1. First Sub-Item
      1. First Sub-Sub Item
      2. Second Sub-Sub Item

A block quote is created with the > symbol at the beginning of a line.

Markdown files allows a table to be constructed in 2 ways, manually or using a package such as the gt package. A table is manually created by using |, :, and -. The first line contains | and the column names in between. The second line contains |:-|:-| which indicates how the table is aligned. The location of : symbol just tells RStudio about the alignment. 15.1 Math

Quarto is capable of writing mathematical formulas using LaTeX code. A mathematical symbol can be written inline using single $ signs. For example, $\alpha$ is viewed as \(\alpha\) in a document. To write mathematical formulas on its own line use $$. For example, $$Y=mX+b$$ is viewed as \[Y=mX+b.\]
